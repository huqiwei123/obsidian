- [ ] 能力指标构建

模型计算的时候根据机场id（airportId）动态传入
#### 作战范围能力

1. 超音速速度
   机场中速度最大的战斗机的速度作为超音速速度
	1. table：飞机表
	2. oper：select max(速度) from 飞机表 where type = '战斗机'
2. 战机最大作战半径
   衡量军事机场的作战半径能力时，取这个机场的最大作战半径的飞机的作战半径作为整个机场的最大作战半径
	1. table：飞机表
	2. oper：select max(作战半径) from 飞机表 where type = '战斗机'
3. 射程（空空导弹能力）
   从导弹表的类型为空空导弹中，找到射程最大的导弹作为该机场的空空导弹射程
	1. table：导弹表
	2. oper：select max(射程) from 导弹表 where type = '空空导弹'
4. 对海打击距离
   从导弹库表中的类型为空地导弹或反舰导弹中，找到对海打击距离最大的空地导弹作为机场的对海打击距离
	1. table：导弹表
	2. oper：select max(对海打击距离) from 导弹表 where type = '空地导弹' and type = '反舰导弹'
5. 对地精打距离
   从导弹库表中的类型为空地导弹或反舰导弹中，找到对地精打距离最大的反舰导弹作为机场的对地精打距离
	1. table：导弹表
	2. oper：select max(对地精打距离) from 导弹库表 where type = '空地导弹' and type = '反舰导弹'
6. 是否有加油机
   机场是否有加油机
	1. table：飞机表
	2. oper：select exists(加油机) from 飞机表 where type = '加油机'
	3. extra_param: {type = '加油机'}
7. 电子战支援半径（**去掉**）

#### 备援接替能力

1. 跑道数量
   机场的所有跑道数（包括备用跑道）
	1. table：跑道表
	2. oper：select count( * ) from 跑道表
2. 跑道长度
   机场主跑道的长度作为跑道长度（默认机场只有一个主跑道）
	1. table：跑道表
	2. oper：select 长度 from 跑道表 where type = '主跑道'
3. 跑道宽度
	机场主跑道的宽度作为跑道端口
	1. table：跑道表
	2. oper：select 宽度 from 跑道表 where type = '主跑道'
4. 道面等级
   道面等级划分为5个等级
	1. table:  跑道表
	2. oper：select 道面等级 from 跑道表 where type = '主跑道'
5. 停机坪规模
   机场中所有停机坪的总面积
	1. table：停机坪表
	2. oper：select sum(面积) from 停机坪表
6. 油库数量
	1. table：油库表
	2. oper：select count( * ) from 油库表
7. 维修设施等级（**去掉**）

#### 出动能力

1. 机场中作战飞机的数量
	1. table：飞机表
	2. oper：select count( * ) from 飞机表 where type = '战斗机'
2. 特种飞机数量（**去掉**）
3. 高强度战时日均出动架次（**去掉**）
4. 单日最大可出动架次（**去掉**）
5. 跑道数
   机场的所有跑道数（包括备用跑道）
	1. table：跑道表
	2. oper：select count( * ) from 跑道表
6. 跑道长度
   机场中主跑道的长度
	1. table：跑道表
	2. oper：select 长度 from 跑道表 where type = '主跑道'
7. 跑道宽度
   机场中主跑道的宽度
	1. table：跑道表
	2. oper：select 宽度 from 跑道表 where type = '主跑道'
8. 主滑行道数量
   滑行道表中存在标记该滑行道类型的字段（0-主滑行道、1-备用滑行道）
	1. table: 滑行道表
	2. oper: count * from 滑行道表 where type = 0
	3. extra_param: {type: 0}
9. 联络道总数量（**去掉**）
10. 快速脱离道数量（**去掉**）
11. 油料存储总量
	1. table：油库表
	2. oper：sum(储油量) from 油库表
12. 加油保障点数据（**改为油库数量**）
	1. table：油库表
	2. oper：count * from 油库表
13. 弹药库总储量
	1. table：弹药库表
	2. oper：select sum(存储量) from 弹药库表
14. 弹药保障设备数量
	1. table: 弹药库表
	2. oper：select sum(弹药保障设备数量) from 弹药库表

#### 打击能力

1. 可用打击飞机数量
	1. table：飞机表
	2. oper：select count( * ) from 飞机表 where type = '战斗机' and type = '轰炸机'
2. 单次平均任务载荷
   取机场所有飞机的载弹量的平均值作为单次平均任务载荷
	1. table：飞机表
	2. oper：select avg(载弹量) from 飞机表 where type = '战斗机' and type = '轰炸机'
3. 单次波最大投送量
	计算所有战斗机的总载荷量作为单波次最大投送量（理论上单次投送量为所有满载的飞机全部出动的投送量）
	1. talbe：飞机表
	2. oper：select sum(载弹量) from 飞机表 where type = '战斗机' and type = '轰炸机'
4. 精确制导弹药储量
	1. table：弹药表
	2. oper：select sum(存储量) from 弹药表 where type = '精确制导弹药'
5. 精确制导弹药占比
	1. table：弹药表
	2. oper：select sum(存储量) from 弹药表 where type = '精确制导弹药' / select sum(存储量) from 弹药表
6. 油库容量
	1. table：油库表
	2. oper：sum(储油量) from 油库表
7. 加油点数量（**改为油库数量**）
	1. table：油库表
	2. oper：count * from 油库表
8. 弹药库容量
	1. table：弹药库表
	2. oper：select sum(存储量) from 弹药库表
9. 弹药保障设备数量
	1. table: 弹药库表
	2. oper：select sum(弹药保障设备数量) from 弹药库表

#### 防空能力

1. 预警雷达种类数（**去除**）
2. 预警雷达数量
	1. table：雷达信息表
	2. oper：count * from 雷达信息表 where type = '预警雷达'
3. 预警雷达的探测距离
   取预警雷达中探测距离最大值作为该机场的预警雷达探测距离
	1. table：雷达信息表
	2. oper：select max(探测距离) from 雷达信息表 where type = '预警雷达'
4. 防空导弹系统种类
	1. table：防空系统表
	2. oper：select count() from 防空系统表
5. 防空导弹系统数量
	1. table：防空系统表
	2. oper：select count( * ) from 防空系统表
6. 防空火炮种类（**去除**）
7. 防空火炮数量
	1. table：防空系统表
	2. oper：select count( * ) from 防空系统表 where type = '防空火炮'
